# 占星对应 (Astrology Correspondences)

## 1. 核心定义
“占星对应”指的是把塔罗牌与占星学中的行星、星座、元素或宫位式时间感建立联系。最常见的做法，是让大阿卡纳对应行星或黄道星座，让小阿卡纳数字牌对应黄道 `36` 个十度区段，再让宫廷牌对应星座的三种模式（本位、固定、变动）。[来源: CTB]

`CTB` 认为，这并不是唯一可能的系统，但却是现代塔罗里最常见、也最成体系的一种解释法。它让牌义不只停留在图像和数字，还能获得一层“宇宙气候”般的附加语感。[来源: CTB]

`YAT` 虽没有像 `CTB` 那样给出完整总表，却不断把四花色与四元素、火象星座等联系起来，并在个别牌中点出具体星座气质，例如把 [权杖骑士](../minor-arcana/wands/knight-of-wands.md) 与射手座特质并读。[来源: YAT]

## 2. 在塔罗体系中的角色
占星对应在塔罗体系中的最大作用，是提供第二层语义压缩。若图像告诉我们“发生了什么”，占星层常会进一步告诉我们这股力量更像哪种节奏、性情或周期。[来源: CTB]

例如 `CTB` 用 [魔术师](../major-arcana/the-magician.md) 对应水星、[皇帝](../major-arcana/the-emperor.md) 对应牡羊座，说明当二者共同出现时，可以像在读“水星进入牡羊”的组合效果，借此强化主动、快速、直接而富开创性的色调。[来源: CTB]

这套系统也常被用来补充时间感。`CTB` 指出，不同行星的运作速度差异，会让同样是“变化即将发生”的判断，带上完全不同的快慢与重量感。不过它也明确提醒：占星对应只是辅助层，而不是解牌的必要前提。[来源: CTB]

## 4. 解读应用实录
在大阿卡纳中，占星对应最适合作为“性格语气”的补充层。`CTB` 列出的示例包括： [愚者](../major-arcana/the-fool.md) 对应天王星、[魔术师](../major-arcana/the-magician.md) 对应水星、[皇帝](../major-arcana/the-emperor.md) 对应牡羊座、[正义](../major-arcana/justice.md) 对应天秤座、[倒吊人](../major-arcana/the-hanged-man.md) 对应海王星。[来源: CTB]

在小阿卡纳数字牌中，`CTB` 采用的是非常典型的“十度区段”思路。每个花色的 `Ace` 对应本花色元素，而 `2-10` 会被分配给同元素的三个星座：火象 [权杖](../concepts/four-elements.md) 落在牡羊 / 狮子 / 射手，水象圣杯落在巨蟹 / 天蝎 / 双鱼，风象宝剑落在天秤 / 水瓶 / 双子，土象星币落在摩羯 / 金牛 / 处女；具体到单牌，又会再叠加行星，例如 `权杖二 = 火星在牡羊`、`圣杯二 = 月亮在巨蟹`、`宝剑二 = 金星在天秤`、`星币二 = 土星在摩羯`。[来源: CTB]

宫廷牌的对应则更偏向“模式”而不是单一行星。`CTB` 把 `Knight / Queen / King` 分别连到变动 / 本位 / 固定星座，因此同属火象的 [权杖骑士](../minor-arcana/wands/knight-of-wands.md) 、[权杖王后](../minor-arcana/wands/queen-of-wands.md) 、[权杖国王](../minor-arcana/wands/king-of-wands.md) 就能分别带出射手、牡羊、狮子的不同火性风格。[来源: CTB] `YAT` 也以类似方式，在个别宫廷牌中直接点出射手等星座气质。[来源: YAT]

## 5. 多源视角
[来源: CTB] 提供的是最系统的版本。它明确讨论了大阿卡纳如何对应行星与星座、小阿卡纳如何映射到 `36` 个十度区段、宫廷牌如何与本位 / 固定 / 变动星座相连，并指出这套系统对组合阅读与时间判断都有帮助。

[来源: YAT] 的写法较为分散，却更贴近日常解读。它把四花色和火 / 水 / 风 / 土元素及相应星座气质挂钩，并在若干牌的解释里直接借用星座个性来加强画面，因此更像一种“会渗进牌义里的补充语言”。[来源: YAT]

综合来看，占星对应最适合作为塔罗的辅助语法，而不是唯一钥匙。`CTB` 给了它完整骨架，`YAT` 让这套骨架更像活生生的性格与节奏描写；二者结合时，它最能发挥价值。
