# 七张牌 (Seven Card)

## 1. 牌阵概述
七张牌牌阵是 `YAT` 中最适合作为“通用主力牌阵”的结构之一。它既能做整体分析，也能回答明确问题，而且比十张以上的大牌阵更容易稳定抓住主线。[来源: YAT]

`YAT` 很看重它的原因，在于它能同时涵盖时间线、问题核心、周遭环境、主观情绪与结果，不会因为信息太少而只剩片段，也不会因为牌太多而把重点冲散。[来源: YAT]

当问题比 [单牌启示](./single.md) 或 [圣三角形](./holy-triangle.md) 更复杂，但又还没复杂到必须动用 [赛尔特十字](./celtic-cross.md) 时，七张牌通常是一个很平衡的选择。[来源: YAT]

## 2. 各位置语义深解
1. **过去**：回溯到过去十八个月左右，显示当前议题的背景与前因。[来源: YAT]
2. **现在**：描述当前状态，涵盖占卜前后约四周的时间感。[来源: YAT]
3. **最近结果**：从当前状态自然延伸出的近程发展，约三个月左右。[来源: YAT]
4. **答案 / 当事人**：在整体分析里代表问卜对象本人；若是针对某个问题，则直接代表问题的答案。[来源: YAT]
5. **周遭能量**：围绕问卜者或问题的外部氛围、他人态度、朋友 / 同僚 / 相关人物的影响。[来源: YAT]
6. **希望与恐惧**：问卜者对局势的期待、顾虑与投射常在这里纠缠出现。[来源: YAT]
7. **结果**：当前模式下的较长程结果，时间通常可延伸到两年以内。[来源: YAT]

## 3. 关键位置间关系与张力模型
这個牌阵最强的第一条线是 `1 -> 2 -> 3` 的时间推进。它不仅告诉你“发生了什么”，也告诉你某个背景是否还在供能、现状是不是已在转弯，以及短期趋势会不会把问题推向另一个阶段。[来源: YAT]

第二条关键线是 `4 -> 7`。若这是问题分析，`4` 给出核心答案，`7` 给出更完整的结果；两者相互呼应时，结论会相当清楚，若彼此拉扯，则说明答案本身仍受条件限制。[来源: YAT]

第三条张力在 `5` 与 `6`。`5` 往往是外界真实存在的气候，`6` 则是问卜者主观上的希望和恐惧。很多解读误差，正是因为把这两者混在一起。[来源: YAT]

## 4. 解读策略与切入点
若是问题分析，最稳的起手通常是先看 `4` 与 `7`，因为这两张最接近用户真正想知道的答案；随后再用 `2`、`5`、`6` 去解释这个答案为什么成立或为什么被拖延。[来源: YAT]

`YAT` 还给出一个很实用的判断法：若是需要倾向性较明确的是 / 否问题，`答案位` 与 `结果位` 若都呈积极、正立或明显支持性的牌，结论就会更稳；若 `答案位` 逆位而 `结果位` 正向，则更像“不是永远不行，而是目前延迟”。[来源: YAT]

它尤其适合关系发展、职业阶段变化、较长周期的现实抉择，以及需要同时看见环境与内心态度的议题。[来源: YAT]

## 5. 常见分析陷阱（失败模式）
最常见的错误，是把 `4` 固定读成某个人，而忘了在问题分析里它可能直接就是答案。[来源: YAT]

第二个错误，是把 `7` 当成宿命终局。`YAT` 写得很清楚，这更像是当前状态延伸出的结果，而不是脱离前面六张牌独立成立的判决。[来源: YAT]

第三个错误，是忽略 `5` 和 `6` 的差别。若把外部氛围误读成主观恐惧，或把内在投射误读成客观事实，整副牌会很容易失真。[来源: YAT]
